Privacy Policy for XDM Auditor

Last updated: 2026

XDM Auditor is a browser extension designed to help developers debug and validate Adobe Experience Platform (AEP) Web SDK (Alloy.js) implementations.

1. Data Collection

XDM Auditor does not collect, store, or transmit any personal data.

2. Data Usage

All data processed by the extension remains locally within your browser. It is used solely for debugging, inspection, and validation of network requests and XDM payloads.

3. Permissions

The extension may request permissions such as access to browser tabs, storage, and network activity (including <all_urls>). These permissions are strictly used to analyze and debug web SDK implementations and are not used to track users or collect personal information.

4. Data Sharing

XDM Auditor does not share any data with third parties.

5. Security

Since no data is collected or transmitted externally, there is no risk of data exposure from this extension.

6. Changes to This Policy

This privacy policy may be updated in the future. Any changes will be reflected on this page.
